Design system · Opzione 3 — Arancio e Technical Design

Due colori, una gerarchia
di intensità.

Stessa targhetta tecnica dell'opzione 2 (mire d'angolo, monospazio per i codici), ma con un solo accento cromatico — l'arancio — invece di quattro. Lo stato di un articolo si legge dall'intensità dell'arancio (contorno → tinta → pieno), non dal colore. Spaziatura interna dei componenti ridotta rispetto alle altre due opzioni; titoli e ritmo tra sezioni invariati.

Da opzione 1 — solo arancio #E88706 e graphite #333333, nessun altro hue
Da opzione 2 — targhetta, mire d'angolo, Big Shoulders/Plex Sans/Plex Mono
Nuovo qui — severità per intensità (non per colore), componenti più densi

Token pronti per Figma (Tokens Studio) e per il CSS della futura web app.

{
  "global": {
    "color": {
      "orange":   { "value": "#E88706" },
      "graphite": { "value": "#333333" }
    },
    "spacing": { "xs": 4, "sm": 8, "md": 12, "lg": 16, "xl": 24, "xxl": 32 }
  }
}

Come portarlo in Figma

  1. Plugin gratuito Tokens Studio for Figma
  2. Import → incolla design/tokens-orange-technical.json
  3. "Create Styles/Variables"
  4. Font: "Big Shoulders Display", "IBM Plex Sans", "IBM Plex Mono" — già su Google Fonts

Le variabili CSS in cima a questo file sono la stessa fonte del token JSON — copiabili come base del foglio di stile.
