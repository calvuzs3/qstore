Design system · v1

Targhetta, non icona.
Ogni articolo è uno strumento.

QuickStore gestisce ricambi elettromeccanici reali — controller ABB, servomotori, cuscinetti — non prodotti da scaffale di supermercato. Questo sistema tratta ogni riga di magazzino come la targhetta identificativa di uno strumento: codice in monospazio, stato leggibile a colpo d'occhio, mire d'angolo come su un disegno tecnico.

Soggetto — inventario ricambi industriali multi-magazzino, foto-riconoscimento OpenCV
Chi lo usa — tecnici in magazzino, da telefono, spesso a mani impegnate
Compito della UI — far leggere giacenza e ubicazione in un'occhiata, rendere i movimenti tra magazzini impossibili da sbagliare

Export

Token pronti per Figma (plugin Tokens Studio) e per il CSS della futura web app.

{
  "global": {
    "color": {
      "graphite": { "value": "#12181F" },
      "paper":    { "value": "#F3F5F6" },
      "teal":     { "value": "#0E6E66" },
      "amber":    { "value": "#E98A2E" },
      "green":    { "value": "#3A8F5C" },
      "red":      { "value": "#C4432E" }
    },
    "spacing": {
      "xs": 4, "sm": 8, "md": 12, "lg": 16, "xl": 24, "xxl": 32
    }
  }
}

Come portarlo in Figma

  1. Installa il plugin gratuito Tokens Studio for Figma
  2. Apri il plugin → Import → incolla il contenuto di design/tokens.json
  3. Crea gli Styles/Variables dai token importati (un click, "Create Styles")
  4. Font: cerca "Big Shoulders Display", "IBM Plex Sans", "IBM Plex Mono" nel menu font di Figma — sono già su Google Fonts, nessun upload richiesto

Riuso per la web app

Le stesse variabili CSS in cima a questo file (:root) sono già pronte da incollare come foglio di stile base — stessi nomi, stessi valori del token JSON, nessuna traduzione da fare.
